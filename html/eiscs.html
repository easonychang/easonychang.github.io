EIS Computer Science

Fall 2016 (2nd - 5th Grade)

Activity 1: Method Calling Sequence

Using the online tool code.org to teach the order that the functions are called through mazes and puzzles.

Throughout the activity, students will be able to use building blocks to move their angry bird character towards the green pig. And these building blocks students are using are actually lines of code.

It is a good way to learn the sequence of codes, meaning which line of code executes first and eventually understand each line of code used when connecting the block is called method calling.

Maze: Sequence(1-11)
Minecraft Hour of Code (1-7)
Week 1 Activity Report

Activity 2: Debugging

Using the online tool code.org to teach debugging through mazes and puzzles.

Throughout the activity, students will be given a set building blocks to move their angry bird character towards the green pig. And these building blocks provided to the students are not necessarily correct.

Students will have to edit the buggy code to arrive at the correct solution.

Maze: Debugging(1-12)
Minecraft Hour of Code (8-14)
Week 2 Activity Report

Activity 3: Loops and Conditionals

Using the online tool code.org to teach basic computer science concepts such as loops and conditionals through mazes and puzzles.

Throughout the activity, students will be introduced new building blocks to move their angry bird character towards the green pig.

The new features (blocks) "if blocks" and "repeat blocks" will help students with to come up with better approaches to tedious tasks. Students will also learn the logic of “if or else” to make answers more efficient.

Maze: Loops(1-14)
Bee: Conditionals (1-15)
